# VBA Footer Docstring

Please review the code placed in the previous prompt and use the following template to create a docstring:

```vba
''===========================================================================================
'' Procedure: ......... [procedure_name]
'' Description: ....... [procedure_description]
'' Version: ........... 1.0.001 - major.minor.patch
'' Created: ........... [YYYY-MM-DD]
'' Updated: ........... n/a
'' Module URL: ........ n/a
'' Installs to: ....... n/a
'' Compatibility: ..... [compatibility with Word, Excel, etc.]
'' Contact Author: .... lundeen-bryan
'' Copyright:  ........ company [copyright symbol] YYYY. All rights reserved.
''
'' Parameters:
'' name(type): param_description
''
'' Returns:
'' (type): param_description
''
'' Preconditions:
'' precondition description text
''
'' Calls To:
'' procedures that are called by this procedure
''
'' Called By:
'' procedures that call this procedure
''
'' Notes:
'' (1) notes here
''
'' Examples:
'' (1) examples here
''===========================================================================================
```
